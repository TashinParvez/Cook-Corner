# Cook Corner

## Overview
Cook Corner is a web application that allows users to explore cooking recipes, manage ingredients, and enhance their culinary skills.

## Features
- Search for recipes and ingredients
- User authentication and profile management
- Responsive design using Bootstrap

## Installation
- Ensure PHP 8.2.0 is installed on your system.
- Include frontend libraries via CDN in your HTML files.

## Usage
- Open the [Homepage.php](https://github.com/TashinParvez/Cook-Corner/blob/main/Home/Homepage.php) file in your web browser.
