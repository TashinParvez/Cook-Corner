## Overview

**Cook Corner** is a full-featured cooking assistant and recipe management web platform built to support users in discovering recipes, planning meals, tracking nutrition, generating dishes from available ingredients, and improving their overall cooking experience.

The platform is designed for:

- home cooks
- food lovers
- health-conscious users
- beginners learning to cook
- users looking for meal-planning support
- people who want to reduce food waste

Rather than being just a recipe website, Cook Corner combines **recipe discovery, nutrition guidance, shopping assistance, occasion-based browsing, meal planning, and cooking education** into one system.

---

## Introduction

Many people struggle with everyday cooking because of problems like:

- not knowing what to cook
- limited ingredients at home
- difficulty planning meals
- maintaining healthy eating habits
- wasting leftover ingredients
- spending extra time shopping

**Cook Corner** was created to solve those problems through an intuitive digital platform that helps users cook smarter and more confidently.

It offers easy recipe exploration, nutrition-aware meal planning, ingredient-based recipe generation, grocery support, and additional learning tools like kitchen tips and cooking courses.

## Motivation

Cook Corner was built to make cooking:

## Core Features

## 1. Recipe Browsing

Users can explore a large collection of recipes based on different categories, meal types, and preferences.  
Each recipe can include ingredients, preparation instructions, preparation time, and nutritional details.

## 2. Recipe Generator

Users can enter available ingredients and receive suggested recipes based on those inputs.  
This encourages creativity and helps reduce food waste.

## 3. Nutrition-Based Suggestions

Cook Corner can recommend recipes according to predefined nutrition plans and calorie-related goals, helping users choose meals that fit their health objectives.

## 4. Nutrition Alert

The platform provides nutrition-related awareness so users can make more informed food decisions and maintain healthier eating habits.

## 5. Occasion-Based Recipes

Users can discover recipes for events, seasonal celebrations, holidays, parties, and special occasions, making the platform more contextual and useful.

## 6. Meal Planning

The system helps users organize meals for a week or month based on personal needs and food preferences.

## 7. Grocery List Support

Users can create and manage grocery lists according to selected recipes so they can prepare efficiently before cooking.

## 8. Ingredient Ordering and Checkout

Cook Corner includes shopping-related functionality where ingredients can be ordered and reviewed through a checkout flow.

## 9. Advanced Search Functionality

The platform supports recipe searching and filtering by ingredients, dietary preferences, meal types, and other cooking-related categories.

## 10. Add New Recipe

Registered users can contribute to the platform by adding their own recipes, helping expand the recipe collection and community value.

## 11. Favorite Recipes

Users can save preferred recipes for quick access later.

## 12. Dietary Preference Support

The platform is designed to support dietary needs such as:

- vegetarian
- vegan
- gluten-free
- low-fat
- heart-healthy
- keto
- and other health-focused options

## 13. Cooking Courses

Users can explore cooking courses to improve their culinary knowledge and practical skills.

## 14. Kitchen Tips

Cook Corner includes kitchen tips that help users improve cooking techniques, efficiency, and confidence in the kitchen.

## 15. Apply for Chef Title

The platform includes a feature where users can apply for recognition based on skill and contributions, encouraging deeper engagement.

## 16. BMI Calculator

Users can calculate BMI by entering height and weight, gaining extra health-related insight alongside nutrition-focused features.

## How It Works

Cook Corner follows a modular PHP web application approach.

### Typical user flow:

1. A user visits the platform
2. The user signs up or logs in
3. The user browses recipes or searches by ingredients
4. The user explores nutrition-aware suggestions or meal plans
5. The user saves favorites or adds recipes
6. The user prepares a grocery list or proceeds to ingredient ordering
7. The user learns through kitchen tips and cooking courses

This creates a full cooking journey from planning to preparation.

## Installation Guide

## Prerequisites

Before running this project locally, make sure you have:

* PHP **7.4 or higher**
* MySQL
* Apache or another local web server
* phpMyAdmin or another database manager
* A local development environment such as:

  * **XAMPP**
  * **WAMP**
  * **Laragon**

The repository also includes Composer configuration and a listed PHP dependency.

---

## Local Setup

### 1. Clone the repository

```bash
git clone https://github.com/TashinParvez/Cook-Corner.git
```

### 2. Move the project into your local server directory

Example:

* **XAMPP:** `htdocs/`
* **WAMP:** `www/`

Example path:

```bash
C:/xampp/htdocs/Cook-Corner
```

### 3. Install PHP dependencies

If Composer is required in your setup:

```bash
composer install
```

### 4. Create the database

Create a MySQL database for the project, for example:

```sql
cook_corner
```

### 5. Import the SQL file

Import the project database SQL file into your MySQL database.
If the SQL file is stored in a project folder, import that file through phpMyAdmin.

### 6. Configure database connection

Open the database connection file used by the project and update the credentials.

Typical local setup may look like:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "cook_corner";
```

### 7. Start Apache and MySQL

Run your local development server environment.

### 8. Open the project in your browser

```bash
http://localhost/Cook-Corner/
```
